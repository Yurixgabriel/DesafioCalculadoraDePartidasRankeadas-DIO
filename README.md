# DesafioCalculadoraDePartidasRankeadas-DIO
## Descrição do Desafio
O desafio consiste em criar uma calculadora de níveis ranqueados para jogadores com base no saldo de vitórias. O objetivo é determinar o nível do jogador com base no número de vitórias e derrotas.

## Variáveis
- vitorias: Número representando a quantidade de vitórias do jogador.
- derrotas: Número representando a quantidade de derrotas do jogador.
- saldoVitorias: Diferença entre o número de vitórias e derrotas.
- ranking: Array que armazenará o saldo de vitórias e o nível correspondente.

## Níveis Ranqueados
- Ferro: Saldo de vitórias até 10.
- Bronze: Saldo de vitórias até 20.
- Prata: Saldo de vitórias até 35.
- Ouro: Saldo de vitórias até 50.
- Diamante: Saldo de vitórias até 70.
- Lendário: Saldo de vitórias até 100.
- Imortal: Saldo de vitórias acima de 100.
